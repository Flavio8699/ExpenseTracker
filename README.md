# ExpenseTracker
 
Expense tracker app build with SwiftUI and Core Data.

# Packages used
1) QGrid (Grid layout): https://github.com/Q-Mobile/QGrid
2) FASwiftUI (FontAwesome icons): https://github.com/mattmaddux/FASwiftUI
